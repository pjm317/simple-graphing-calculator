# README

## Setup

1. Install [Python](https://www.python.org/downloads/).

2. Install the requirements:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```

## Ideas

Feel free to fork the app and modify it.

Some ideas include polar, parametric, and 3D graphs:

- [Matplotlib polar plots](https://matplotlib.org/stable/api/_as_gen/matplotlib.pyplot.polar.html)
- [SymPy plotting](https://docs.sympy.org/latest/modules/plotting.html)
- [Matplotlib 3D examples](https://matplotlib.org/stable/gallery/mplot3d/index.html)
